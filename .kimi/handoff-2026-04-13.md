### Task 2: Wire ConstructionContainer to Infrastructure Blocks ‚úÖ
**File modified:** `app/containers/construction.py`

Added `_get_or_create_cache_key(file_path, doc_type)` helper that:
- Uses `file_hasher` SHA256 when available
- Falls back to `mtime+size` or deterministic path hash for missing files

Rewired `process_document()` to:
1. **Cache lookup** ‚Äî checks `cache_manager` via `action=get`; returns cached result immediately on hit with `_source: "cache"`
2. **File metadata** ‚Äî uses `file_hasher.metadata` to get `file_size`
3. **Async offload** ‚Äî if `file_size > 10MB`, submits to `async_processor` (`task_name="block:construction.process_document"`) and returns `{"status": "queued", "_source": "async_queue"}`
4. **Normal processing** ‚Äî routes to existing doc-type processor (drawing, spec, contract, etc.)
5. **LLM enhancement** ‚Äî on success, calls `llm_enhancer` with `action="structure_json"` and appends result under `llm_enhanced`
6. **Cache write** ‚Äî stores final result in `cache_manager` with TTL `7200` seconds
7. **Metadata tagging** ‚Äî adds `_cache_key` and `_source: "processor"` to result

Also added `get_actions()` method exposing all 39 handler methods.

**Circular import fix:** `BLOCK_REGISTRY` is imported locally inside methods, not at module top-level.
